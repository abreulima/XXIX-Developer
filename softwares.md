# Softwares

[Visual Studio 2026](https://visualstudio.microsoft.com/downloads/) - Microsoft's dedicated Integrated Development Environment (IDE) for .NET Development.

Attention, solutions made using Visual Studio 2026, **will not** open in Visual Studio 2022. Visual Studio is Windows only. For other systems, Rider is recommended.

[Rider](https://www.jetbrains.com/rider/) Jetbrain's dedicated IDE for .Net Development. Cross-Platform working on Windows, macOS and Linux. Rider is free for non-commercial use.
